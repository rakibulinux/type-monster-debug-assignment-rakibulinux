Difference between localStorage and sessionStorage

LocalStorage data does not expire, whereas sessionStorage data is cleared when the page session ends.
A unique page session gets created once a document is loaded in a browser tab.
Page sessions are valid for only one tab at a time.

Difference between global scope and local scope

JavaScript variables have only two scopes Global && Local
Global Variables: A global variable has a global scope which means it can be defined anywhere in your JavaScript code.
A variable declared at the top of a program or outside of a function is considered a global scope variable.
Local Variables: A local variable will be visible only within a function where it is defined.
Function parameters are always local to that function.
